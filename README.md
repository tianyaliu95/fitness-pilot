# Fitness Pilot

Minimal Next.js + TypeScript skeleton prepared for deployment on Vercel.

Quickstart

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
# open http://localhost:3000
```

3. Build for production

```bash
npm run build
npm run start
```

Deploying to Vercel

- Import this repository into Vercel (https://vercel.com/new)
- Vercel will detect Next.js automatically. Build command is `npm run build`.

Notes

- An API route is available at `/api/hello`.
- This branch is `init/vercel-setup` and contains only a minimal structure. Add features (Tailwind, Prisma, Auth) as needed.
